Canada Revenue Agency

Canada Revenue Agency is a department of the Canada government aimed to Administer tax laws for most provinces and territories.

Problem Statement

The Canada Revenue Agency has a confusing navigation system resulting user frustration.How might we make the website aesthetically pleasing, increase user satisfaction by streamlining the navigation, and ultimately create a better user experience.

Solution

Redesign the home page by integrating new UI components

My Role:UI designer

Tools:

Figma, Invision

Usability Study

According to the User Path I conduct two usability tests through desktop and mobile and discovered that :

Users quickly get confused and lost while navigating through the website.

Recommendations

Based on the Usability Study and Heuristic Evaluation results:

  • Add a second navigation menu specifically for the CRA.

  • Make the buttons and Icons consistent.

  • Use breadcrumbs and headers to clearly identify the user's location on the website and help them go back to the previous pages without difficulty.

  • All external links should include appropriate warnings and etiquette information for users.
